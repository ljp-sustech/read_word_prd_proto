# PC 智慧读文原型设计记录

## 目标

基于现有 PC 端系统截图，制作一个用于领导评审展示、兼顾前端交互对齐的静态 HTML 原型。

## 范围

* 仅制作 PC 端原型。
* 使用 `PC_prototype/PC端背景.png` 作为系统背景。
* 在现有系统右下角增加“小机器人/智慧读文”悬浮入口。
* 展示智慧读文助手面板，包括文件上传、意图反问、全文总结、文档问答、猜你想问、结果流转等关键状态。

## 设计

* 原型采用单文件 `PC_prototype/index.html`，纯 HTML/CSS/JS，可直接双击打开。
* 画面默认展示现有 PC 系统底图和右下角机器人入口。
* 点击机器人入口后打开右侧助手面板。
* 面板内使用顶部状态切换按钮，便于评审时快速演示不同状态。
* 视觉风格沿用政务系统蓝色调，采用白底、浅蓝边框和轻阴影，避免与原系统割裂。

## 实现计划

1. 创建 `PC_prototype/index.html`。
2. 引用 `PC端背景.png` 作为全屏背景图。
3. 绘制右下角机器人悬浮入口。
4. 绘制智慧读文助手面板。
5. 增加状态切换：初始引导、意图反问、全文总结、文档问答、结果流转。
6. 补充猜你想问按钮和结果操作按钮。
7. 校验 HTML 文件存在、图片引用存在、关键文案存在。
